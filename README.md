📦 Getting Started

1. Download the template package.
2. Extract the ZIP file.
3. Open the project folder in your preferred code editor.
4. Customize the HTML, Tailwind CSS, and Alpine.js files to match your brand and content.
5. Upload the files to your web hosting server or deploy them using GitHub Pages.
